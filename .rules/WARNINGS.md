# Warnings Log

Security and integrity warnings are logged here automatically.
Review this file regularly to monitor system health.

## Severity Levels

- **CRITICAL**: Immediate attention required (immutable rules compromised)
- **PROTOCOL_VIOLATION**: Session started without proper initialization
- **SESSION_EXPIRED**: Session marker too old
- **MERGE_INTEGRITY_CHANGE**: Protected files changed during merge
- **WARNING**: General warnings
